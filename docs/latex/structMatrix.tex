\doxysubsection{Detailed Description}
Основная структура для представления матрицы в памяти. 

Структура содержит указатель на двумерный массив данных и размерности матрицы. Все операции библиотеки работают с этой структурой.

\begin{DoxyNote}{Note}
Память для data должна выделяться динамически с помощью create\+\_\+matrix 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
Обязательно освобождать память с помощью free\+\_\+matrix для избежания утечек. 
\end{DoxyWarning}
